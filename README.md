# Errachidia (ERH) Terminal Monitor

Real-time flight arrival tracking system and automatic Telegram notifier for Moulay Ali Cherif Airport, Errachidia (ERH) Moroccan Aerodrome.

## Features

- Real-time flight arrival tracking
- Automatic Telegram notifications
- Professional slate and steel aesthetic
- Arabic language support
- Firebase authentication

## Setup

1. Clone this repository
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` and fill in your credentials
4. Run development server: `npm run dev`

## Technologies

- React + TypeScript
- Vite
- Firebase
- Motion (Framer Motion)
